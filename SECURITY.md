# Security Policy

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Email **info@clientreach.ai** with:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

We'll respond within 48 hours and aim to ship a fix within 7 days for confirmed issues.

## Scope

This policy covers the `particleui-cli` npm package and the `particleui.dev` web app. Component source files installed into your own project are your responsibility once copied.
