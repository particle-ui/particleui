# Introduction

ParticleUI is a registry of premium animated components built on top of [shadcn/ui](https://ui.shadcn.com).
You install components directly into your project using the **ParticleUI CLI**. The source lands in
your repo, so you can customize it like normal code.

## Installation

### 1. Install a free component

Start with Button. No sign-in or token is required:

```bash
npx particleui-cli add button
```

The component source lands in `src/components/ui/button.tsx`.

### 2. Optional: configure your registry

If you also use shadcn-compatible registries directly, add the `@particleui` registry to your
`components.json`:

```json title="components.json"
{
  "registries": {
    "@particleui": {
      "url": "https://particleui.dev/r/react/{name}.json",
      "headers": {
        "Authorization": "Bearer ${PARTICLEUI_TOKEN}"
      }
    }
  }
}
```

### 3. Add your token for Pro components

```bash title=".env"
PARTICLEUI_TOKEN=your-token-here
```

Get a token from your [dashboard](/dashboard) after signing up.

### 4. Install another component

```bash
npx particleui-cli add glow-button
```

Free components work without a token. Pro components require a valid `PARTICLEUI_TOKEN`.

## Frameworks

| Framework | Registry URL |
|-----------|-------------|
| React | `https://particleui.dev/r/react/{name}.json` |
| Vue | `https://particleui.dev/r/vue/{name}.json` *(coming soon)* |
| Svelte | `https://particleui.dev/r/svelte/{name}.json` *(coming soon)* |
